
**1. 提供店铺域名**
需要提供店铺域名给UseePay 运营，域名格式为xxx.myshopline.com，提供后待UseePay 运营配置好后再进行安装applepay/googlepay，否则无法搜索。


![image.png](https://api.apifox.com/api/v1/projects/6993167/resources/568335/image-preview)

点击店铺后台的“域名”界面


![image.png](https://api.apifox.com/api/v1/projects/6993167/resources/568336/image-preview)
找到“主域名”
**2. 绑定店铺**
（1）运营通知配置完成后，您可以在店铺的  设置 -> 收款-其他收款方式  进行勾选

![image.png](https://api.apifox.com/api/v1/projects/6993167/resources/568337/image-preview)
点击店铺后台的“收款”界面

![image.png](https://api.apifox.com/api/v1/projects/6993167/resources/568338/image-preview)
其他收款方式处点击“添加”
**注意：点击对应的支付方式进行安装，如需同时安装applepay和googlepay，2个都需要安装**

![image.png](https://api.apifox.com/api/v1/projects/6993167/resources/568339/image-preview)

搜索“UseePay”
**（2）点击后，会跳转到绑定页面，点击安装（Apple pay安装方式跟Google pay相同）**


![image.png](https://api.apifox.com/api/v1/projects/6993167/resources/568340/image-preview)
点击安装
**（3）点击同意隐私条款，点击授权并安装**

![image.png](https://api.apifox.com/api/v1/projects/6993167/resources/568341/image-preview)
同意条款并授权安装
（4）随后会被重定向到MC商户登录页面，输入登录邮箱、操作员名称、密码等信息登录
**注意：如果您有多个商户号，请注意区分并登录正确的商户号进行安装**


![image.png](https://api.apifox.com/api/v1/projects/6993167/resources/568342/image-preview)

登录MC后台账号
**（5）成功登录后，点击右侧下拉列表，选择需要绑定的网站（在UseePay审核通过的网站），点击保存**


![image.png](https://api.apifox.com/api/v1/projects/6993167/resources/568343/image-preview)
选择需要绑定的域名并保存
****（6）点击添加，完成激活，绑定流程完成****

![image.png](https://api.apifox.com/api/v1/projects/6993167/resources/568344/image-preview)
选择添加
